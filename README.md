## BUET | Department of Civil Engineering | Assignment: Class Test 1

## 📌 Overview

This repository contains all model outputs, saved model files, and diagnostic
figures produced for the **CE457 Weekday Trip Generation Regression Assignment**.

The goal was to develop regression models that predict the **total number of
daily weekday trips** (`WD Trip`) generated by BUET students, using socio-economic
variables from the 2025 CE457 Student Travel Survey.

The final strategy (Strategy 5) processed the **raw original dataset** directly,
performed comprehensive data cleaning, stratified train-test splitting, and
benchmarked **44 model configurations** alongside an exhaustive search of
**2,497 OLS variable combinations**.

## ⚙️ Methodology Summary

### Data Processing
- **Source:** `2025_CE457_tgData_v2.xlsx` (69 respondents, 196 columns)
- Irrational values corrected (e.g., `hrs_with_fam` > 24 → NaN)
- Text-encoded numerics converted (e.g., `"more than 10"` → 10)
- 18 leaky/outcome variables excluded (WD sub-components, weekend trips)
- 22 zero-variance columns dropped
- Missing values imputed with **training-set medians** (no data leakage)
- 3 composite features engineered (`total_workers`, `total_students`, `active_members`)

### Train-Test Split
- **Stratified 80/20 split** to preserve target distribution
- Training: n = 56 | Test: n = 13
- Test target (`WD Trip`) preserved in original untouched form

### Model Selection
- Correlation-based ranking of 84 candidate features
- Exhaustive search of **2,497 OLS variable combinations** (top 12 features, sizes 2–6)
- 44 total model configurations benchmarked
- Best model selected by **Test R²** on held-out stratified test set
